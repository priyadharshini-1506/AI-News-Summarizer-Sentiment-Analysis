# AI-News-Summarizer-Sentiment-Analysis
# 📰 AI News Summarizer with Sentiment Analysis

## 📌 Project Overview

AI News Summarizer is an intelligent desktop application developed using Python that automatically extracts and summarizes news articles from a given URL. The application also performs sentiment analysis to determine whether the article conveys a positive, negative, or neutral sentiment.

The application provides a simple graphical user interface (GUI) built with Tkinter, enabling users to obtain article insights with a single click.

## 🎯 Objectives

* Automatically extract news articles from URLs.
* Generate concise summaries of lengthy news content.
* Identify the sentiment of news articles.
* Provide a user-friendly desktop interface.

## ✨ Features

* Extracts article title automatically.
* Retrieves author information.
* Displays publication date.
* Generates AI-based summaries of articles.
* Performs sentiment analysis on article content.
* Classifies sentiment as:

  * Positive 😊
  * Negative 😞
  * Neutral 😐
* Interactive GUI using Tkinter.

## 🛠️ Technologies Used

* Python
* Tkinter
* Newspaper3k
* TextBlob
* NLTK

## ⚙️ Working Process

1. User enters the URL of a news article.
2. The application downloads and parses the article using Newspaper3k.
3. Natural Language Processing techniques generate a concise summary.
4. TextBlob analyzes the article text to determine sentiment polarity.
5. The application displays:

   * Article Title
   * Author(s)
   * Publication Date
   * Summary
   * Sentiment Analysis Result
